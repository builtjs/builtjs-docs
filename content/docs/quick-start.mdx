Let's start making modular Next.js website themes and plugins and use the [Built Studio](/built-studio) to build sites from them!


The easiest way to get started is by running this to create your own theme:

```bash copy
npx create-built-app new theme
```
Then you can follow the steps in the `README` to create an app. The theme will be created using your own AI model.

<Callout type="info">
This method of creating a Built.js theme requires an OpenAI API key. If you prefer to to create a theme manually, you can follow [this guide](/docs/guides/how-to-create-a-theme).
</Callout>

If you just want to get a feel for how Built.js works, you can:

<Steps>
<Step>
### Create a theme
 
To create a theme, you can either use the [Built Studio](https://builtjs.com) Theme Designer or you can clone an existing theme:
```bash copy
git clone https://github.com/richjava/corporate-shadcn.git
```
Follow the directions in the project's README to set up and run the theme.
</Step>
<Step>
### Publish the theme
 
To be able to use your theme to create sites with in [Built Studio](https://builtjs.com), run this:
```bash copy
npx create-built-app publish
```

</Step>
<Step>
### Create a site
 
In [Built Studio](https://builtjs.com), select the theme you published, choose the modules for the site, and export the site. You can follow the README to set up and run the site.
</Step>
</Steps>
