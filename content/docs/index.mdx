**Built.is** is a platform for creating and managing reusable website themes and plugins for quick and easy creation of CMS-backed [Next.JS](https://nextjs.org) websites. 

Make modular themes and plugins in Javascript and use the [Built Studio](/built-studio) to build sites from them. 
 
If you just want to get a feel for how Built.js works, you can:

<Steps>
<Step>
### Create a theme
 
To create a theme, you can either use the [Built Studio](https://builtjs.com) Theme Designer or you can clone an existing theme:
```bash copy
git clone https://github.com/richjava/corporate-basic.git
```
Follow the directions in the project's README to set up and run the theme.
</Step>
<Step>
### Publish the theme
 
To be able to use your theme to create sites with in [Built Studio](https://builtjs.com), run this:
```bash copy
npx create-built-app publish
```
</Step>
<Step>
### Create a site
 
In [Built Studio](https://builtjs.com), select the theme you published, choose the modules for the site, and export the site. You can follow the README to set up and run the site.
</Step>

</Steps>
