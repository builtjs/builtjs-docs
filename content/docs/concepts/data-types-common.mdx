# Common data types

The data types available for a Built.js [Content Type](/docs/concepts/content-type), [Global](/docs/concepts/global), [Element](/docs/concepts/element), and [Section](/docs/concepts/section) data are:

## `biginteger`

| Property | Description                     | Required |
| -------- | ------------------------------- | -------- |
| type     | The data type                   | true     |
| required | Determines if value is required. Default: `false` | false |
