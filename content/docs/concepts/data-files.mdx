# Data files

Data files contain JSON data used for configuration of theme or plugin's pages, sections, plugins and more. They are kept in ```public/data```.  

These are the folders and files that you will find in this directory for a theme:

<Callout type="info">
  For a plugin, you will find a `plugin.json` file instead of `theme.json`.
</Callout>

## Schema files
Schema files are for configuring the fields, or "schema" for Collections, Elements, Global data, and Sections.

## Collection data files
Collection data files are named with the plural name of the content type, like ```authors.json``` and each file contains data for entries in the collection.

## Main data files
These files contain the main data for the theme/plugin, such as module data.

Learn more about the Data Model defined by these files in the [Data Model Reference](/data-model-reference).
