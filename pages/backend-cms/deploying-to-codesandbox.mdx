## How to deploy your Strapi CMS to CodeSandbox

<Callout emoji="⚠️">
  **Attention** In many cases CodeSandbox may not be suitable for production sites
</Callout>

1. In the `README.md` document, you'll see a button that looks like this:

![Deploy to CodeSandbox](/images/codesandbox-btn.png "Deploy to CodeSandbox")

Click it to deploy your project to CodeSandbox.

2. In the project editor of CodeSandbox, go to "Server Control Panel (in the left sidenav) > Secret Keys". Enter "ADMIN_JWT_SECRET" into the "Name" field, and a secret value for the "Value" field", and then click the "Add Secret" button.

3. Wait for project to build. If you see a "502: Bad Gateway" message in the browser preview, ignore it. Wait until you see the "Opening the admin panel..." message in the terminal. In a new browser window, go to the url you see in the browser preview address bar, e.g:
```https://<project-name>.sse.codesandbox.io```

4. Go to the Strapi Admin (`/admin`) page and register as a new user.
