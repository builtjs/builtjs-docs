# Built Studio

You can use [Built Studio](https://studio.builtjs.com/) to manage your themes and plugins, create and export sites, and use community themes and plugins.

## Workspace
Your workspace is where you manage your themes and plugins.

### Personal space
The personal space uses the theme and plugin projects that are imported from your personal Github account.

### Team space
A Team space uses theme and plugin projects that are imported from a Github organisation.

<Callout type="info">
  Please note that the purpose of a team workspace is to allow you to manage your Github organisation themes and plugins, and is currently only available to individual users.
</Callout>

## Community Space
The themes and plugins in the community space are available for all to use. Just click on a theme and start creating sites with it. You can also access and add any plugin from the community when creating a theme using the Theme Designer, .

## Site Designer
This is where you create sites. To get to the Site Designer, click on a theme in your workspace or the community space.
To configure a site to your needs, you can choose the modules you want for the site as well as the pages and section templates.

## Theme/Plugin Designer
You can use the Theme/Plugin Designer to create themes or plugins for export. After using the designer, the theme or plugin will have everything it needs so that all you need to do is create the react components for the section templates.
