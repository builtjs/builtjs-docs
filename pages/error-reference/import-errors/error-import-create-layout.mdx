# An error occurred while creating layout for import

During theme or plugin import, layout data is created from data files in the theme/plugin's Github repo. There may be an issue with the data file.

Please check the data file for layout data located at:

<FileTree>
  <FileTree.Folder name="public" defaultOpen>
    <FileTree.Folder name="data" defaultOpen>
      <FileTree.File name="..." />
      <FileTree.File name="layout.json" />
      <FileTree.File name="..." />
    </FileTree.Folder>
  </FileTree.Folder>
</FileTree>

The data model reference for layout is located at [/data-model-reference/#layout](/data-model-reference/#layout).

To learn more, please refer to the [Layout](/concepts/layout) page of the [Concepts](/concepts) documentation.
