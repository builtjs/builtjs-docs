# An error occurred while creating imported content types

During theme or plugin import, content types are created from data files in the theme/plugin's Github repo. There may be an issue with the data file.

Please check the data file for content types located at:

<FileTree>
  <FileTree.Folder name="public" defaultOpen>
    <FileTree.Folder name="data" defaultOpen>
      <FileTree.Folder name="schemas" defaultOpen>
        <FileTree.File name="..." />
        <FileTree.File name="content-types.json" />
        <FileTree.File name="..." />
      </FileTree.Folder>
    </FileTree.Folder>
  </FileTree.Folder>
</FileTree>

The data model reference for content types is located at [/data-model-reference/#content-types](/data-model-reference/#content-types).

To learn more, please refer to the [ContentType](/concepts/content-type) page of the [Concepts](/concepts) documentation.
