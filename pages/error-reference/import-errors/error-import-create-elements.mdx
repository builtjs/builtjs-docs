# An error occurred while creating imported elements

During theme or plugin import, elements are created from data files in the theme/plugin's Github repo. There may be an issue with the data file.

Please check the data file for elements located at:

<FileTree>
  <FileTree.Folder name="public" defaultOpen>
    <FileTree.Folder name="data" defaultOpen>
      <FileTree.Folder name="schemas" defaultOpen>
        <FileTree.File name="..." />
        <FileTree.File name="elements.json" />
        <FileTree.File name="..." />
      </FileTree.Folder>
    </FileTree.Folder>
  </FileTree.Folder>
</FileTree>

The data model reference for elements is located at [/data-model-reference/#elements](/data-model-reference/#elements).

To learn more, please refer to the [Element](/concepts/element) page of the [Concepts](/concepts) documentation.
