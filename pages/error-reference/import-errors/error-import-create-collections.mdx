# An error occurred while creating collections for import

During theme or plugin import, collections are created from data files in the theme/plugin's Github repo. There may be an issue with a collection data file.

If you receive this error, please check the collection data files for located at:

<FileTree>
  <FileTree.Folder name="public" defaultOpen>
    <FileTree.Folder name="data" defaultOpen>
      <FileTree.Folder name="collections" defaultOpen>
        <FileTree.File name="..." />
      </FileTree.Folder>
    </FileTree.Folder>
  </FileTree.Folder>
</FileTree>

The data model reference for collections is located at [/data-model-reference/#collections](/data-model-reference/#collections).

To learn more, please refer to the [Collection](/concepts/collection) page of the [Concepts](/concepts) documentation.
