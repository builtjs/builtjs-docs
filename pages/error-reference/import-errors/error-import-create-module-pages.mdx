# An error occurred while creating module pages for import

During theme or plugin import, module page data is created from data files in the theme/plugin's Github repo. There may be an issue with the data file.

Please check the data file for module page data located at:

<FileTree>
  <FileTree.Folder name="public" defaultOpen>
    <FileTree.Folder name="data" defaultOpen>
      <FileTree.File name="..." />
      <FileTree.File name="module-pages.json" />
      <FileTree.File name="..." />
    </FileTree.Folder>
  </FileTree.Folder>
</FileTree>

The data model reference for module pages is located at [/data-model-reference/#module-pages](/data-model-reference/#module-pages).

To learn more, please refer to the [Module Page](/concepts/module-page) page of the [Concepts](/concepts) documentation.
