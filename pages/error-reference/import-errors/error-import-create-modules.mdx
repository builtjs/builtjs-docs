# An error occurred while creating modules for import

During theme or plugin import, module data is created from data files in the theme/plugin's Github repo. There may be an issue with the data file.

Please check the data file for module data located at:

<FileTree>
  <FileTree.Folder name="public" defaultOpen>
    <FileTree.Folder name="data" defaultOpen>
      <FileTree.File name="..." />
      <FileTree.File name="modules.json" />
      <FileTree.File name="..." />
    </FileTree.Folder>
  </FileTree.Folder>
</FileTree>

The data model reference for modules is located at [/data-model-reference/#modules](/data-model-reference/#modules).

To learn more, please refer to the [Module](/concepts/module) page of the [Concepts](/concepts) documentation.
