# An error occurred while creating pages for import

During theme or plugin import, page data is created from data files in the theme/plugin's Github repo. There may be an issue with the data file.

Please check the data file for page data located at:

<FileTree>
  <FileTree.Folder name="public" defaultOpen>
    <FileTree.Folder name="data" defaultOpen>
      <FileTree.File name="..." />
      <FileTree.File name="pages.json" />
      <FileTree.File name="..." />
    </FileTree.Folder>
  </FileTree.Folder>
</FileTree>

The data model reference for pages is located at [/data-model-reference/#pages](/data-model-reference/#pages).

To learn more, please refer to the [Page](/concepts/page) page of the [Concepts](/concepts) documentation.
