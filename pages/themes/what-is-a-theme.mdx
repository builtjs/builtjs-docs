# What is a Theme?

![Theme](/images/builtjs-vis.png)

A Built.js Theme is a configurable template site that allows you to create a customized website with accompanying content management system. Sites are created from themes in the [Built.JS Studio](https://studio.builtjs.com). Themes include settings which makes it possible for users to design their site by business use case.

A Theme is simply a [Next.js](https://nextjs.org) project hosted on Github. Here is an example:

Code: [https://github.com/builtjs/builtjs-theme-corporate-tailwind](https://github.com/builtjs/builtjs-theme-corporate-tailwind)<br/>

Along with regular Next.JS pages and React components, a theme project contains:
- A set of ```data``` files in the ```public``` directory
- A ```.theme``` folder
- Built.JS NPM libraries

### Data files
Data files are used for configuration of:
- Theme data
- Site data
- CMS data

More information about this can be found in the [Data Model Reference](/themes/data-model-reference).
