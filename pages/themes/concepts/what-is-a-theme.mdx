# What is a Theme?

![Theme](/images/builtjs-vis.png)

A Built.js Theme is what allows a user to create a Site with accompanying content management system. Themes are used by the [Built.JS Studio](https://studio.builtjs.com) and include settings which makes it possible for users to choose parts of the Theme for their Site and CMS.

A Theme is a [Next.js](https://nextjs.org) project hosted on Github. Here is an example:

Code: [https://github.com/builtjs/builtjs-theme-corporate-tailwind](https://github.com/builtjs/builtjs-theme-corporate-tailwind)<br/>
Demo: Coming soon

Along with regular Next.JS pages and React components, a theme project contains:
- A set of ```data``` files
- A set of ```settings``` files

Pages in a Built.JS theme are very basic. This is because, amongst other configurations, the page layout is managed in the ```settings``` files.

### Data files
Data in the ```data``` files is used for:
- Theme data
- Static Site data
- Backend CMS data

### Settings files
Data in the ```settings``` files is used for:
- Theme configuration
- Site configuration
- CMS configuration
