# Introduction 

**Built.JS** is a platform for creating and managing reusable website themes and plugins for quick and easy creation of CMS-backed [Next.JS](https://nextjs.org) websites. 

Make modular themes in Javascript, store them on Github, and use the [Built.JS Studio](https://studio.builtjs.com) to build sites from them. 
 
## Getting started
To familiarize yourself with how Built.js works, check out the [Concepts](/concept-overview), [Guides](/guides-overview) and the [Data Reference](/data-reference).
