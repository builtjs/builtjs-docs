# Getting Started
The frontend project created for you by Built.js is a regular [Next.js](https://nextjs.org) project. 

The project is as opinionated as the Theme that it comes from. Some "vanilla" themes are as unopinionated as possible, whereas others come with a frontend framework (like Tailwind) and other dependencies.

A frontend project can use data in two ways:
- From local files
- From an external CMS

## How to run locally
1. Install the dependencies:
```
npm i
```
2. Run the app in develop mode:
```
npm run dev
```

### Connect to the CMS
The CMS is also generated by Built.JS and setup information is available [here](/backend-cms/getting-started).
