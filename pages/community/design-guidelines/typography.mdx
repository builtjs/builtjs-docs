# Typography
## Display
Display styles are the largest text on the screen, reserved for short, important text. They use a more expressive font, and are usually used sparingly. There are 4 sizes for Display.

| Token           | Example Size | Example Line Height |
| --------------- | ------------ | ------------------- |
| `display-01`    | 36px         | 44px                |
| `display-02`    | 44px         | 52px                |
| `display-03`    | 52px         | 64px                |
| `display-04`    | 96px         | 112px               |
