# Built.JS

**Built.JS** is a platform for creating Javascript themes and CMS-backed [Next.JS](https://nextjs.org) websites. 
<br/>
<br/>
<iframe width="100%" height="515" src="https://www.youtube.com/embed/8_ovzMFyzuw" title="YouTube video player" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>

Make modular themes in Javascript, store them on Github, and use the [Built.JS Studio](https://studio.builtjs.com) to import your themes and build sites from them.
