# Updating a theme or plugin

Once you've imported your theme or plugin into Built Studio, the central source of truth is the theme/plugin's Github repository. It's important to keep your repository in sync with the Studio theme/plugin, so whenever you push changes to the repo or merge updates to a branch, you should sync with the repo. 

You can sync a theme/plugin's main branch by clicking on the card's ellipsis and selecting "Sync with repo". If you want to sync with another branch, you need to switch to the branch in the Studio on the Modules page, and then on this page you can click on the ellipsis in the top bar and select "Sync with repo".

You can't access the Theme/Plugin Designer to make updates to a theme or plugin after it has been imported.
