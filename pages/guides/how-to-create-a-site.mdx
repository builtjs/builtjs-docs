# How to create a site

You can create a site an with a community theme at any time, and once you have your own themes in the studio, you can create sites with them too. The basic process for this is:

<Steps>
### Select a theme
 
Select a community theme or one of your own.
 
### Select modules you need for your site
 
These modules will determine the features of your site.

### Configure your site pages

Choose the section templates you want, delete any pages and sections you don't want.

### Export the site

This will download a zip file to your device.

### Go through the steps in the README to setup and run the site

Learn more about this process in [how to set up a site](/guides/how-to-set-up-a-site).

</Steps>


You are then ready to make any changes you want or directly deploy the site on a host of your choosing.
