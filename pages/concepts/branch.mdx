# Branch

When you import your theme or plugin into the [Built Studio](/built-studio), you can choose the git branch you want to import. 
After import, you can add more branches. This is especially helpful if your theme or plugin is being used by the community. 

The main branch is the only branch that community users can view, so you can import your dev branch into the studio and test it out until you're satisfied before merging the changes into the main branch in Github, and then importing the main branch to update it for the community.
