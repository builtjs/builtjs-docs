# Data files

Data files contain JSON data used for configuration of theme or plugin's pages, sections, plugins and more. They are kept in ```public/data```.

## Schema files
These files are for configuring the fields, or "schema" for Collections, Elements, Global data, and Sections.

## Collection data files
These are named with the plural name of the content type, like ```authors.json``` and each file contains data for entries in the collection.

## Main data files
These files contain the main data for the theme/plugin, such as Module data.
