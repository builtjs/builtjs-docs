# Section data types

The data types available for a Built.js [Section](/concepts/section) are all the [common data types](/concepts/data-types-common) plus the following:

## `element`
An [Element](/concepts/elements) represents a reusable UI component. 

| Property | Description                     | Required |
| -------- | ------------------------------- | -------- |
| type     | The data type                   | true     |
| element  | Name of the element                   | true     |
| repeatable  | Whether the value is single or multiple values. Default: `false` | false |
| required | Determines if value is required. Default: `false` | false |
