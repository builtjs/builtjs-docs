# Common data types

The data types available for a Built.js [Content Type](/concepts/content-type), [Global](/concepts/global), and [Element](/concepts/element) data are:

<Callout type="info">
  Please note that Section and Content Type data types have extra data types. See [Section data types](/concepts/data-types-section) and [Content Type data types](/concepts/data-types-content-type).
</Callout>

| -------- | ------------------------------- | -------- |

| required | Determines if value is required. Default: `false` | false |
